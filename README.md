Overview:

Register Form is a front-end form validation project designed to ensure accurate and properly formatted user input before submission. The project focuses on client-side validation, user feedback, and form handling using vanilla JavaScript.

This application does not include backend submission and is intended to demonstrate front-end validation logic and UX considerations.

Features:

Required field validation

Email format validation

Password confirmation checks

Real-time user feedback

Clean and simple form layout
